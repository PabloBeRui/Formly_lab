Global Extensions (Automatización)

Nota: En este JSON no hay clases de CSS, no hay Badges HTML, ni mensajes de error personalizados. Todo lo que ves es inyectado por la exampleExtension.
Importante: Si no ves cambios (bordes rojos o fondo amarillo), asegúrate de descomentar la línea extensions: [...] en tu archivo app.config.ts.
Observarás:
  • Mutación de Labels: El sufijo "(extension)" se inyecta en los campos requeridos.
  • Ayuda Dinámica: Descripción extra: "Campo obligatorio extension".
  • Detección de Tipo: El input se vuelve amarillo y el checkbox resalta en rojo.
  • Bordes Globales: Un borde rojo envuelve a todo elemento required.
⚡ Alcance Global: Esta extensión actúa como un "interceptor" de Formly. Debido a que está registrada en el app.config, afectará a todos los componentes de la aplicación que usen Formly, asegurando una coherencia visual absoluta en todo el proyecto.
